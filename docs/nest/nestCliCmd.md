# 🐔 Nest CLI 常用指令

## 建立 project

```shell
nest new [project-name]
```

## 建立 controller

```shell
nest generate controller [module name]
```

## 建立 service

```shell
nest generate service [module name]
```

## 建立 module

```shell
nest generate module [module name]
```

### 縮寫

- nest generate 縮寫

```shell
nest g
```
