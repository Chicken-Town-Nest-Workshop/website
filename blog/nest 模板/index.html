2023年9月27日 · 閱讀時間約 1 分鐘

nest js 模板終於上線啦!

模板內容​

  1. swagger
  2. dockerfile

使用方式​

下載​

載點

  • code > download Zip
  • 解壓縮

模板使用方式​

  • 修改專案名稱
  • 於 package.js 修改 name
{
  "name": "[專案名稱]",
  "version": "0.0.1",
  "description": "",
  "author": "",
  "private": true,
  "license": "UNLICENSED",
  "scripts": {
    "build": "nest build",
-- 下略 --
  • 安裝依賴
pnpm i
  • 啟動
pnpm start

開啟: http://localhost:3055/api

docker​

建立 image​

docker build --pull --rm -f ".dockerfile" -t [image name]:[version] .

run image​

docker run --name nest-app -p 3055:3055 [image name]:[version]

開啟: http://localhost:3055/api
